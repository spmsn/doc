## Qu'est-ce qu'un opérateur ?

Comme son nom l'indique, il permet de faire une opération, par exemple mathématique.

```python
# Affiche 4
print(2 + 2)

# Un calcul un peu plus compliqué
print((6 + 2) - (6 + 3))
```

## Opérateurs courants

- Les additions et soustractions : `+` et `-`
- Les multiplications et divisions : `*` et `/`
- Les comparaisons qu'on verra plus loin : `==`, `!=`, `>`, `>=`, `<`, `<=`

:::note
Il existe [plein d'autres opérateurs](https://www.w3schools.com/python/python_operators.asp) à utiliser au besoin.
:::
